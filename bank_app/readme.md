# Mini Bank App (FastAPI + HTML)

A simple banking demo with:
- User authentication
- Bank transfer
- Deposit feature
- HTML + JS frontend
- FastAPI backend

## Run FastAPI
uvicorn main:app --reload

## Open UI
Open bank.html in your browser
